# Cpp-StratumV2

Implementation of the Stratum V2 protocol in C++.  
This project focuses on the cryptographic handshake and secure communication layer.

Stratum V2 uses a complex key exchange mechanism conceptually similar to TLS, involving multiple steps of negotiation, verification, and secure state establishment.  
The implementation includes logic for building shared keys and validating certificates during the handshake process.

Originally developed as part of production work, with emphasis on correctness, low-level control, and protocol understanding.
